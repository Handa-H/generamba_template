## Overview
It's catalog of templates for Generamba.

## How to use
1. Add this catalog to your Rambafile.

```
### Templates
catalogs:
- 'https://github.com/Handa-H/generamba_template/'
templates:
- {name: vp_of_mvp}
```

2. Run `generamba template install`.

3. Run `gemeramba gen [MODULE NAME] vp_of_mvp`.
